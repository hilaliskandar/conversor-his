# Roadmap

## Marco 1 — fundação

- estrutura modular;
- CLI;
- diagnóstico de PDF;
- extração nativa;
- OCR seletivo com Tesseract;
- manifesto e hash;
- métricas CER e WER.

## Marco 2 — qualidade documental

- amostragem estratificada;
- elementos críticos;
- fidelidade estrutural;
- thresholds configuráveis;
- relatório de erros;
- testes com documentos reais.

## Marco 3 — motores concorrentes

- OCRmyPDF;
- PaddleOCR;
- benchmark entre configurações;
- seleção automática por página.

## Marco 4 — Markdown normativo

- artigos;
- parágrafos;
- incisos;
- alíneas;
- tabelas;
- anexos;
- rastreabilidade por página.

## Marco 5 — empacotamento

- instalador Windows;
- documentação operacional;
- GitHub Actions;
- release versionada.
